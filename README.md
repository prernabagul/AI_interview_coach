# AI Interview Coach

An AI-powered interview preparation platform built using Streamlit, Gemini AI, SQLite, and PDF Resume Parsing.

## Features

- Upload Resume (PDF)
- Resume Parsing using PDFPlumber
- AI-Generated Interview Questions
- Difficulty Levels (Easy, Medium, Hard)
- AI-Based Interview Evaluation
- Interview History Tracking
- SQLite Database Integration

## Tech Stack

- Python
- Streamlit
- Gemini AI
- SQLite
- PDFPlumber

## Installation

```bash
pip install -r requirements.txt
streamlit run Interview.py
```
